# Maincrafts-Cybersecurity-Internship-Task2
A hands-on cybersecurity lab built during my internship at Maincrafts Technology. Features a Kali Linux attacker VM and OWASP Juice Shop vulnerable web application deployed via Docker, connected over an isolated Host-Only network in VirtualBox. Includes Nmap scanning, Burp Suite traffic interception, and Wireshark packet capture.
